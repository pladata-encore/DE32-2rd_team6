## 실행 방법
```bash
$ git clone https://github.com/pladata-encore/DE32-2rd_team6
$ cd DE32-2rd_team6
$ source .venv/bin/activate
$ pip install 
$ python src/de32_2rd_team6/con_chat3.py
$ python src/de32_2rd_team6/pro_chat3.py
```
